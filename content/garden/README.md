# 数字花园 · 节点内容

「星际之门 · 数字花园」的想法节点。每个 `.md` 文件是星图里的一个节点，
非线性、可互链，按成长阶段生长。

## 新增一个节点

1. 复制 `_template.md`，重命名为有意义的英文短名（如 `kv-cache.md`）——**文件名就是节点 id**。
2. 填写 frontmatter：
   - `title`：显示标题；
   - `stage`：`seedling`（🌱 萌芽）/ `budding`（🌿 生长）/ `evergreen`（🌲 常青），决定节点颜色；
   - `tags`：标签；
   - `links`：**相邻节点的文件名数组**（去掉 `.md`），星图据此连线；
   - `updated`：更新日期，用于排序。
3. 正文用 Markdown 书写，可用 `[[文件名]]` 提示互链（连线以 `links` 为准）。

## 约定

- 文件名以 `_` 开头的会被忽略（如模板）。
- `draft: true` 的节点仅本地可见，线上隐藏；发布时删掉该行。
- 连线是无向的：A `links` 到 B，即便 B 没回链 A，星图也会画出这条边。
