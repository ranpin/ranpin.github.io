## 个人网站项目概况
- **项目名称**: ranpin-portfolio (个人学术作品集网站)
- **仓库地址**: https://github.com/ranpin/ranpin.github.io
- **在线预览**: https://ranpin.github.io/
- **核心功能**: 响应式设计、深色/浅色主题、内容管理系统、管理面板、Markdown编辑器、智能搜索、版本管理、自动备份、数据验证、批量操作、媒体文件支持、自定义分类标签
- **组件数量**: 20个核心组件 (AdminPanel, ArticleList, AutoBackup, BatchOperations, DataValidation, DateRangePicker, Footer, GuestBook, Header, Hero, LazyImage, MarkdownEditor, MediaViewer, Profile, ProjectTemplates, SearchBox, SectionTitleEditor, SmartRecommendations, TagCloud, VersionManager)
- **数据源**: src/data/content.js (LocalStorage持久化)
- **部署方式**: GitHub Actions自动构建部署到GitHub Pages
